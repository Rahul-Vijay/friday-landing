Features

Built like a workshop, not a text editor.

Friday isn't where you write code line by line — it's the room your agents work in while you direct them.

01

Infinite canvas

Pan and zoom across an endless workspace. Every terminal, browser, and note gets room to breathe — no more digging through stacked tabs.

02

Parallel agents

Run Claude Code, Codex, OpenCode, Gemini CLI, Cursor CLI, Aider and Amp side by side — same repo, same canvas, zero context switching.

03

Live browser pane

A real, driveable browser your agents can navigate, click through, and read from — not a screenshot, an actual session sitting right on the canvas.

04

Persistent memory

Every project remembers its own context across sessions. Pick up exactly where you and your agents left off.

05

Sticky notes & whiteboard

Sketch a plan, pin a note, draw a line between two agents. The canvas is a whiteboard first, a terminal grid second.

06 ⌘K

Command bar

Summon everything with a keystroke: spawn an agent, jump panes, run a saved command. Built for people who don't reach for the mouse.
