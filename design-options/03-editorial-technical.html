Section 04 — Questions

Before you ask.

The four questions we get every week, answered plainly.

Q1

Is Friday a code editor?

No. Friday is a harness — a canvas that hosts terminal panes running your existing CLI agents, plus a browser and notes. Keep using whatever editor you already use.

Q2

Do I need my own API keys?

Yes — Friday is bring-your-own-key by design. You already pay for Claude Code, Codex or Gemini CLI directly; Friday just orchestrates them, with nothing routed through our servers.

Q3

What does "lifetime" actually cover?

Every future update to Friday, on every platform we ship, for as long as the product exists — one license, purchased once, no renewal ever required.

Q4

What about Linux?

In active development. Buy today on macOS or Windows and the Linux build is included in your license at no extra cost when it ships.
