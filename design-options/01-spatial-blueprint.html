Spec sheet

Questions, answered plainly

01What exactly is Friday?

Friday is an AI agent harness: a desktop app that turns a spatial canvas into a workspace for running AI coding agents. Place terminals, a browser, and notes side by side — Friday wires it together with persistent memory, so your agents work like a real team.

02Which tools work with Friday?

Anything that runs in a terminal — Claude Code, Codex, Gemini CLI, Cursor CLI, OpenCode, Amp, Aider, your own scripts. Friday hosts the panes and you keep using the agents you already trust.

03Do I bring my own API keys?

Yes. Friday is local-first and never proxies your traffic. Your API keys and your code stay on your machine, connected once and available to every agent pane.

04Is Friday a code editor?

No — it's the workspace around your editor. Keep editing in Zed, VS Code, or whatever you love. Friday handles the running agents, the browsing, and the memory.

05How is this different from opening terminals?

Terminals forget everything, don't share context, and can't browse. Friday adds spatial layout, a real browser agents can drive, a whiteboard, and memory that survives across sessions.
